# Estudos em Linguagem C

Este repositório contém estudos aprofundados na linguagem de programação C, abrangendo tópicos como ponteiros, alocação de memória, structs, manipulação de arquivos, pré-processador, funções de biblioteca padrão, ponteiros para funções, programação concorrente, depuração, alguns conceitos de recursividades e tambem listas (pilhas e filas).
